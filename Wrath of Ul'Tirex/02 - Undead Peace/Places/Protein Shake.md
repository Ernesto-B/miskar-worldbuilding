# Description
The commoner save version of the [[Holy Protein Shake]]

# Effect
After consuming the Shake, User gets advantage on the next **Athletics** and **Acrobatics** check.

Consuming 100 of these will grant the user immunity from the poison damage the [[Holy Protein Shake]] would take.
